## 第一部分：Obsidian 是什么？

### 什么是 Obsidian？

Obsidian 是一款强大的本地知识管理工具，它将你的笔记存储为纯文本 Markdown 文件，完全由你掌控。与云笔记应用不同，Obsidian 的核心理念是：

- **本地优先**：所有数据存储在你的电脑上，不依赖云服务
- **隐私安全**：没有服务器，没有数据上传，完全私密
- **永久可用**：即使 Obsidian 公司停止运营，你的笔记仍然是可读的 Markdown 文件
- **高度可定制**：通过插件和主题，打造属于你的知识管理系统

### Obsidian 的核心特性

1. **双向链接（Backlinks）**：笔记之间可以相互关联，形成知识网络
2. **图谱视图（Graph View）**：可视化你的知识结构
3. **强大的搜索**：快速查找任何笔记内容
4. **插件生态**：社区开发的数百个插件扩展功能
5. **主题系统**：自定义界面外观

#### 详细对比分析

**vs Notion**
- **Obsidian 优势**：本地存储、隐私更安全、离线可用、数据完全属于你
- **Notion 优势**：开箱即用、协作功能强、云端自动同步、学习资源丰富
- **适用场景**：Obsidian 适合个人知识管理；Notion 适合团队协作和复杂数据库

**vs OneNote**
- **Obsidian 优势**：双向链接、知识网络、插件丰富、Markdown 原生支持
- **OneNote 优势**：与 Office 集成、手写支持、学习曲线平缓
- **适用场景**：Obsidian 适合知识管理；OneNote 适合笔记记录和学生使用

**vs Evernote**
- **Obsidian 优势**：本地存储、隐私安全、双向链接、完全免费
- **Evernote 优势**：云端同步、搜索功能强、Web 剪藏工具
- **适用场景**：Obsidian 适合长期知识积累；Evernote 适合快速信息收集

**vs Apple Notes**
- **Obsidian 优势**：双向链接、知识网络、插件丰富、跨平台
- **Apple Notes 优势**：与 Apple 生态集成、简洁易用、协作功能
- **适用场景**：Obsidian 适合深度知识管理；Apple Notes 适合快速记录


#### 为什么选择 Obsidian？

**如果你关心以下几点，Obsidian 是最佳选择：**

1. **隐私和数据安全** - 数据完全本地存储，无需担心隐私泄露
2. **长期可用性** - Markdown 格式永久可读，不依赖任何服务
3. **知识网络构建** - 双向链接和图谱视图帮助建立知识体系
4. **完全掌控** - 可以自定义一切，支持 Git 版本控制
5. **成本效益** - 完全免费，无隐藏费用
6. **离线工作** - 无需网络连接，随时随地工作
7. **学习和研究** - 特别适合学生、研究人员、知识工作者

**不适合 Obsidian 的场景：**
- 需要实时团队协作
- 需要自动云端同步（可通过配置实现，但需要额外设置）
- 希望开箱即用，不想配置
- 需要手写笔记功能

## 第三部分：快速开始

### 第一步：安装 Obsidian

1. 访问 [Obsidian 官网](https://obsidian.md)
2. 下载对应系统版本（Windows/Mac/Linux）
3. 安装并打开应用

### 第二步：创建你的第一个库（Vault）
#### 方式1:
1. 打开 Obsidian，选择"创建新库"
2. 选择一个本地文件夹存储笔记
3. 建议放在云同步文件夹（如 iCloud Drive、OneDrive）以实现多设备同步
#### 方式2:
1. 打开 GitHub，创建一个私有repository
2. 克隆git 仓库到本地， 可以是云同步文件夹
3. 打开Obsidian，打开本地库即可

## 第五部分（续）：创建高效的笔记模板

模板是提高笔记效率的关键工具。通过使用 Templater 插件，你可以创建动态模板，自动填充日期、变量等信息。以下是基于实际使用的模板创建指南。

### 为什么需要模板？

- **节省时间**：无需每次手动输入重复的结构
- **保持一致**：确保同类型笔记的格式统一

let pages = dv.pages(`"Reviews/Daily"`)
// 使用当前日期
const now = new Date();
const year = now.getFullYear();
const month = now.getMonth() + 1; // JavaScript月份从0开始，所以+1

const habits = {
	'vocabulary': '☑️ 背{habit}个单词',

	'workout':'☑️ 运动',

let data = {}
for (let page of pages) {
	let date = page.file.name
	data[date] = data[date] || ''
	for (let task of page.file.tasks.filter(task => task.tags.contains(habit_tag) && task.checked)) {

			if (task[habit]) {
				data[date] += habits[habit].replace('{habit}', task[habit]) + '\n'
			}
		}
	} 
}

let calendarData = []
for (let date in data) {

### 如何使用模板？

#### 方法 1：使用 Templater 插件

1. 在 Templates 文件夹中创建模板文件
2. 在设置中配置 Templater 的模板文件夹路径
3. 创建新笔记时，使用命令面板选择"Templater: Create new note from template"
4. 选择相应的模板

#### 方法 2：使用 QuickAdd 插件

1. 安装 QuickAdd 插件
2. 创建快捷命令指向模板
3. 使用快捷键快速创建笔记

#### 方法 3：使用 Periodic Notes 插件

1. 配置 Periodic Notes 的模板路径
2. 自动为日记、周记、月记创建笔记

### 模板最佳实践

1. **保持简洁**：模板不要过于复杂，只包含必要的部分
2. **使用变量**：充分利用 Templater 的变量功能自动填充信息
3. **添加说明**：在模板中添加注释说明每个部分的用途
4. **定期优化**：根据实际使用情况调整模板
5. **创建多个版本**：为不同场景创建不同的模板

## 第七部分：常见问题

### Q1：Obsidian 和其他笔记应用的区别？
**A**：Obsidian 强调本地存储、隐私安全和知识网络。你的数据完全属于你，可以随时导出。

### Q2：如何开始使用 Obsidian？
**A**：
1. 安装 Obsidian
2. 创建一个库
3. 启用推荐插件
4. 创建第一篇日记
5. 逐步建立你的系统

### Q3：插件太多会影响性能吗？
**A**：不会。只安装你需要的插件。建议从 5-10 个核心插件开始。

### Q4：如何备份笔记？
**A**：
- 放在云同步文件夹（iCloud、OneDrive）
- 使用 Obsidian Git 推送到 GitHub
- 定期导出为 HTML 或 PDF

### Q5：可以在手机上使用 Obsidian 吗？
**A**：可以。Obsidian 有 iOS 和 Android 版本，通过云同步实现多设备访问。

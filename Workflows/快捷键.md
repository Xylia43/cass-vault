### Periodic Note
```
Open daily note: control + option + d
Open weekly note: control + option + w
Open monthly note: control + option + m
Open yearly note: control + option + y
```
### QuickAdd 
```
Run QuickAdd： command + option + q
Capture to daily note： command + option + a
```

### **Natural Language Dates**
Insert the current date and time： shift+option+d
Insert the current time： shift+option + t
